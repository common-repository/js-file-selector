Select Javascript files and/or write Javascript functions to any single page or post

== Description ==
A simple Wordpress plugin that allows users to select files Javascript and/or write Javascript functions to any single page or post

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload entire `js-file-selector` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it, you'll see a `Select Javascript files and/or write your Javascript functions` metabox on all add or edit posts/pages.

== Frequently Asked Questions ==

= How to include script functions in the footer =

To include script functions in the footer, the theme should have the wp_footer() template tag in the appropriate place.
